# Contributing to Arch-Render AI Toolkit 🏗️

First off, thank you for considering contributing to this project!

### How Can I Contribute?
* **Reporting Bugs**: Use the GitHub Issues tab to report any bugs you find.
* **Suggesting Enhancements**: We are looking for better ways to parse architectural data (IFC, OBJ, Rhino).
* **Pull Requests**: We follow a standard "branch-and-merge" workflow. Please create a feature branch before submitting a PR.

### Development Process
1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. Ensure the code follows Python PEP 8 standards.
4. Issue that PR!
